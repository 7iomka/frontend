Design tokens

Design tokens are named entities that store visual design information. These are used in place of hard-coded values (such as hex values for color or pixels for spacing) in order to maintain a scalable, consistent system for UI development.

Borders

Width and radii tokens to style element borders.

Variable Value
$border-width-thin 1px
$border-width-thick 2px
$border-width-thicker 4px
$border-width-thickest 8px
$border-radius-small 0.125rem
$border-radius-default 0.25rem
$border-radius-circle 50%

Breakpoints

Breakpoint tokens for use within @media queries.

Variable Value
$mq-min 280px
$mq-xsmall 360px
$mq-small 480px
$mq-medium 504px
$mq-large 800px
$mq-xlarge 960px
$mq-max 1280px

Font families

Font family tokens for typographic styling.

Variable Value
$font-family-sans ‘Source Sans Pro’, sans-serif
$font-family-serif ‘Source Serif Pro’, serif
$font-family-monospace ‘Source Code Pro’, monospace
$font-family-system -apple-system, BlinkMacSystemFont, ‘Segoe UI’, Roboto, Oxygen, Ubuntu, Cantarell, ‘Open Sans’, ‘Helvetica Neue’, sans-serif

Layers

Layering tokens to set the z-index layer value for elements.

Variable Value
$z-index-modal 9000
$z-index-backdrop 8000
$z-index-overlay 1000
$z-index-default 1
$z-index-underlay -1

Sizes

Sizing tokens for describing the dimensions of elements.

Variable Value
$size-xsmall 0.25rem
$size-small 0.75rem
$size-medium 1.5rem
$size-large 3rem
$size-xlarge 4.5rem

Spacing

Spacing tokens for describing the distance between elements.

Variable Value
$space-xsmall 0.25rem
$space-small 0.5rem
$space-medium 1rem
$space-large 1.5rem
$space-xlarge 3rem
$space-xxlarge 4.5rem
